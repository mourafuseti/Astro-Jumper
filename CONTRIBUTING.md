# Contributing to Astro Jumper

Obrigado por querer contribuir! Seguem diretrizes rápidas para facilitar colaborações.

1. Fork o repositório e faça mudanças em uma branch separada
   - Nome da branch: `feature/<descrição>`, `fix/<descrição>` ou `docs/<descrição>`.

2. Código e estilo
   - Use Python 3.8+ compatível com o projeto.
   - Mantenha o estilo simples e legível. Siga a PEP8 quando possível.
   - Testes: adicione testes pequenos quando corrigir bugs críticos ou adicionar features.

3. Commits
   - Mensagens curtas e descritivas: `Add <feature>`, `Fix <bug>`, `Update README`.
   - Faça múltiplos commits separados por passos lógicos.

4. Pull Requests
   - Abra PR para a branch `main` do repositório original.
   - Descreva o que foi feito, por que e qualquer impacto conhecido.
   - Aguarde revisão; responda a comentários rapidamente.

5. Recursos (assets)
   - Se adicionar imagens, sons ou músicas, verifique as licenças.
   - Prefira incluir fontes e assets livres de uso ou com permissões claras.

6. Execução local
   - Siga o `README.md` para configurar ambiente virtual e dependências.

7. Questões legais
   - Ao enviar contribuições você concorda em licenciar suas mudanças sob a licença do projeto (MIT, conforme `LICENSE`).

Obrigado por contribuir — qualquer ajuda é bem-vinda!
